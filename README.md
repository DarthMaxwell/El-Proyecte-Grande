# ISM - El Proyecte Grande

A full-stack team project. The application was built across 4 sprints using Agile methodology. 

We have built a movie blog platform where users can:

- Register and log in
- Create posts about specific movies
- Comment on posts
- View posts created by specific users
- View posts created by specific movies
- Edit/delete your own posts/comments

## Requirements
To run the application, you need:
- Docker
- Docker Compose

## How to run
1. Clone repository from https://github.com/DarthMaxwell/El-Proyecte-Grande
2. From the project directory in a terminal, run: 

```bash
   docker-compose up
```
3. Access the application at http://localhost:3000
